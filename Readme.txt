This is a SnapChat bot for Android mobiles.

For this Bot to work on your PC you'll have to have following things.

1) An Android phone.
2) A usb cable to connect phone to the PC.
3) After connecting your phone with the PC you need to turn developer options On. And also, turn usb debbuging on.
4) In your PC you have to install Git bash for windows.
   https://git-scm.com/downloads
5) Install adb (Android Debug Bridge).
   https://forum.xda-developers.com/showthread.php?t=2588979

NOTE: Don't update your snapchat because updating snapchat will require to change the code too.
